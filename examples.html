Examples

Real-world usage patterns.

Plain HTML

<div data-wh></div>
<script src="https://unpkg.com/webhanger-front/browser.min.js"></script>
<script>
  WebHangerFront.load(CDN_URL, PROJECT_ID, TOKEN, 0);
</script>

React

import { load } from "webhanger-front";
useEffect(() => { load(CDN_URL, PROJECT_ID, TOKEN, 0, "#nav"); }, []);

With signal preloader

WebHangerFront.load(CDN_URL, PROJECT_ID, TOKEN, 0, "[data-wh]", ({ stage }) => {
  if (stage === "start") showSpinner();
  if (stage === "done")  hideSpinner();
});
